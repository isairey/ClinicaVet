## 🐶 Descripción

Sistema de gestión para **clínicas veterinarias**, desarrollado principalmente en **Java**, orientado a la administración de mascotas, clientes, citas médicas y procesos clínicos.

El sistema permite llevar un control eficiente de la información médica y administrativa en un entorno organizado y funcional.

---

## 🚀 Características

- 🐾 Registro de mascotas  
- 👤 Gestión de propietarios  
- 📅 Administración de citas  
- 🩺 Historial clínico  
- 💊 Control de tratamientos  
- 📋 Registro de consultas  
- 📊 Reportes del sistema  
- 🔐 Gestión básica de usuarios  

---

## 🧩 Módulos del Sistema

- 🐶 **Mascotas** → Registro de pacientes  
- 👤 **Clientes** → Información de propietarios  
- 📅 **Citas** → Agendamiento de consultas  
- 🩺 **Historial Clínico** → Diagnósticos y tratamientos  
- 💰 **Administración** → Control general  

---

## 🛠️ Tecnologías Utilizadas

- ☕ Java (Lógica del sistema)  
- 🎨 CSS (Estilos)  
- ⚡ JavaScript (Interactividad)
